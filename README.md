## 소개/기능
비디오, 사진, 할 일, 메모를 한 공간에 저장해둘 수 있는 사이트
  - 비디오/사진은 url을 입력받고 할 일/메모는 text를 입력받음
  - 삭제 기능
  - 드래그 앤 그랍 구현
